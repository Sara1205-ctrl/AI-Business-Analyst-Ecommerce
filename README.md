# AI Business Analyst for E-commerce

## Overview

This project is an AI-powered Business Analytics Dashboard built using Python, Streamlit, Pandas, and Google Gemini AI.

Users can upload e-commerce sales data, analyze KPIs, visualize trends, and receive AI-generated business insights.

## Features

* CSV Upload
* Data Preview
* KPI Dashboard
* Sales by Category Analysis
* Sales by Region Analysis
* Monthly Sales Trend
* Manager Question Input
* AI-Powered Business Insights using Gemini AI

## Technologies Used

* Python
* Pandas
* Streamlit
* Google Gemini AI

## Business Value

The application helps managers make data-driven decisions by converting raw sales data into meaningful insights and recommendations.
